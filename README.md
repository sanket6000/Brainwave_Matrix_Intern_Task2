# DealMart E-Commerce Website
DealMart is a simple and responsive e-commerce website built using HTML, CSS, and JavaScript. It allows users to browse products, add them to their cart, and manage their wishlist. The website supports user login functionality and persists data using localStorage.

**Features**
1. Product Display: View a variety of products with their details (name, price, image).
2. Cart Functionality: Add items to the cart, view the cart, and remove items.
3. Wishlist: Save favorite items to a wishlist for later.
4. User Login: Simple user login system to authenticate and maintain session.
5. Responsive Design: The website is fully responsive and works on both desktop and mobile devices.
6. LocalStorage: Uses localStorage to persist cart, wishlist, and user login data across sessions.

**Tech Stack**
HTML: Used for structuring the content and layout of the pages.
CSS: Styling and responsive design with Flexbox and CSS Grid.
JavaScript: Handles the logic for adding/removing items from the cart, user login, and localStorage.
